### [Cartography](https://github.com/robb/Cartography)

##### In short, it allows you to replace this:
```Swift
addConstraint(NSLayoutConstraint(
    item: button1,
    attribute: .Right,
    relatedBy: .Equal,
    toItem: button2,
    attribute: .Left,
    multiplier: 1.0,
    constant: -12.0
))
```
##### with this
```Swift
constrain(button1, button2) { button1, button2 in
    button1.right == button2.left - 12
}
```
